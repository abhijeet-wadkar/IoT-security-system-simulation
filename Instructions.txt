Instruction to build the code:
1) Open the command prompt
3) Give executable permission to build.sh 
	$ chmod +x compile.sh
4) executed compile.sh (./compile.sh) script
	$ ./compile.sh

Running the System
1) Running using script (run.sh)
	$ chmod +x run.sh
	$ ./run.sh

2) Running Manually
- command to run gateway_front_tier
	$ cd obj/gateway/front_tier
	$ ./gateway_front_tier ../../SampleConfigurationFiles/SampleGatewayConfigurationFile.txt

- command to run gateway_back_tier
	$ cd obj/gateway/back_tier
	$ ./gateway_back_tier ../../SampleConfigurationFiles/SampleGatewayConfigurationFile.txt ../../SampleConfigurationFiles/SamplePersistentStorageFile.txt

- command to run door sensor
	$ cd obj/door_sensor
	$ ./door_sensor ../../SampleConfigurationFiles/SampleSensorConfigurationFile2.txt ../../SampleConfigurationFiles/SampleDoorSensorInputFile.txt

- command to run motion sensor
	$ cd obj/motion_sensor
	$ ./motion_sensor ../../SampleConfigurationFiles/SampleSensorConfigurationFile1.txt ../../SampleConfigurationFiles/SampleSensorInputFile1.txt

- command to run key chain sensor
	$ cd obj/key_chain_sensor
	$ ./key_chain_sensor ../../SampleConfigurationFiles/SampleSensorConfigurationFile3.txt ../../SampleConfigurationFiles/SampleSensorInputFile2.txt

- command to run smart device
	$ cd obj/device
	$ ./device ../../SampleConfigurationFiles/SampleSmartDeviceConfigurationFile1.txt
